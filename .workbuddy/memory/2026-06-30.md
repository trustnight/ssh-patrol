## 修复截图历史"打开文件夹"按钮闪烁

**问题**: 点击"打开文件夹"按钮后文件管理器闪烁但不打开文件夹。
**根因**: `subprocess.Popen(['explorer', screen_dir])` 两个问题：
  1. 路径含中文（巡检助手），subprocess 编码可能乱码
  2. 已有 explorer 进程时新进程会与现有窗口冲突导致闪烁
**修复**: `screenshot.py` 改用 `os.startfile(screen_dir)` — Windows 原生打开文件夹的方式，正确支持中文路径，不会产生进程冲突。同时移除不再使用的 `import subprocess`。

## 长截图"未滚动"根因修复 (6/30)

**问题**: 长截图 Alt+W 按了完全没反应，不滚动也截不到图。

**两个根因**:
1. **pynput 双 GlobalHotKeys 冲突** — Windows 上 pynput 底层使用同一个低级键盘钩子，创建两个独立 `GlobalHotKeys` 监听器时第二个会被覆盖/忽略，Alt+W 根本没被注册成功
2. **焦点问题** — `keybd_event` 发送 Page Down 时目标窗口不一定有焦点，Page Down 可能发送到别的窗口去了

**修复 (screenshot_service.py)**:
- 合并两个 `GlobalHotKeys` 为单个，一个 dict 同时映射 Alt+Q 和 Alt+W
- 新增 `_get_foreground_window()` / `_set_foreground_window()` — 截图前记录浏览器窗口 HWND，每轮滚动前强制恢复焦点
- `_send_page_down()` 新增 `target_hwnd` 参数，优先用 `PostMessageW` 发送 WM_KEYDOWN 到目标窗口（不依赖键盘焦点）
- 补回不慎丢失的 `_get_screen_dir()` 方法
- 滚动后等待从 150ms 增加到 200ms（确保渲染完成）

## 简化浏览器窗口聚焦逻辑 (6/30 第二轮)

**问题**: URL 跳转后浏览器窗口被缩小（最大化→还原成小窗口），且 `_bring_browser_to_front` 实现过于复杂（EnumWindows 遍历标题匹配 IP）。

**根因**: `_bring_browser_to_front` 和 `_set_foreground_window` 中无条件调用 `ShowWindow(hwnd, SW_RESTORE)` — 如果浏览器是最大化状态，`SW_RESTORE` 会将其还原成非最大化的小窗口。

**修复 (screenshot_service.py)**:
- `_set_foreground_window(hwnd)` — 新增 `IsIconic(hwnd)` 检查，仅窗口最小化时才 `SW_RESTORE`，最大化状态保持不变
- `_bring_browser_to_front()` — 大幅精简：移除 EnumWindows 遍历+标题匹配逻辑，直接 `GetForegroundWindow()` 取前台窗口然后 `SetForegroundWindow`。同样仅最小化时恢复
- 移除不再使用的 `SW_SHOW` 常量

## 长截图滚动不生效修复 (6/30 第三轮)

**问题**: 按 Alt+W 长截图，结果保存的是单张普通截图，没有滚动拼接。

**根因**: `_send_page_down(target_hwnd)` 优先使用 `PostMessageW` 发送 `WM_KEYDOWN` 到浏览器窗口。但现代浏览器（Chrome/Edge）不响应 `PostMessage` 发送的键盘消息，导致 Page Down 根本没触发滚动。连续两张截图完全一样，`_images_similar` avg_diff=0 < 5，循环立即 break，只保存了第一张。

**修复 (screenshot_service.py)**:
- `_send_page_down()` — 删除 `PostMessage` 分支，改为直接 `keybd_event` 全局发送。循环中已调用 `_set_foreground_window` 确保焦点在浏览器上，keybd_event 能正确发送给浏览器
- `_calc_diff(img1, img2)` — 新增，提取差异计算逻辑，供日志输出和 `_images_similar` 复用
- 循环等待时间从 `0.2s` 增加到 `0.3s`（确保浏览器滚动动画完成）
- 每帧增加差异值日志 `print(f"[截图服务] 第 {i+1} 帧差异: {diff:.2f}")`，方便调试
